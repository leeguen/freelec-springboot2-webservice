# 스프링 부트와 AWS로 혼자 구현하는 웹 서비스 클론 코딩

## 프로젝트 설명
- 스프링 부트와 AWS로 혼자 구현하는 웹 서비스( 저자 : 이동욱)책을 공부하며 따라서 만들어본 프로젝트

## 개발환경
- 운영체제 : window 10
- IDE : Intellij
- jdk : jdk 11
- DB : MariaDb
- build tool : gradle

## Dependency
- Spring Web
- Lombok
- jpa
- h2
- mariadb
- spring session
- spring oauth2-client
- mustache


## 구현 기능
- social 로그인(naver,google*: google은 정책변경으로 작동x)
- 로그아웃
- 게시글 삭제
- 게시글 작성
- RDBMS 사용
- amazone 서버에 배포
